# 更新日志

## v0.2.5 - 修复命令超时残留进程

- 为命令模板创建独立进程组，避免超时或截断后遗留子孙进程
- POSIX 下改为终止整个进程组，Windows 下改为终止整个进程树
- 新增更新日志文件，记录版本变更

## v0.2.4 - 限制模板读取内存占用

- 文件模板改为按块读取，达到上限后停止
- 命令模板改为流式读取输出，达到上限后停止
- 新增 `read_chunk_size` 配置项，可调整单次读取块大小

## v0.2.3 - 增加注入日志

- 注入模板时输出简短日志
- 日志区分 prompt 占位符注入和自动追加注入

## v0.2.2 - 改进命令模板工作目录

- `custom_workdir` 覆盖 `workdir_base`
- `custom_workdir` 留空时直接使用基准目录
- 路径错误时额外显示允许的目录范围

## v0.2.1 - 调整命令模板执行方式

- 命令模板改为自动拆分参数后执行
- 新增 `executable` 与 `args` 配置项
